## Flujo de Usuario

### Registro de Usuario:
1. El usuario completa un formulario con su información personal.
2. Se valida la información ingresada.
3. Si la validación es exitosa, se crea una nueva cuenta de usuario.
4. Si hay errores de validación, se muestran mensajes de error.

### Inicio de Sesión:
1. El usuario ingresa su correo electrónico y contraseña.
2. Se verifican las credenciales.
3. Si son válidas, se inicia sesión y se redirige al usuario.
4. Si son inválidas, se muestra un mensaje de error.

## Proceso de Registro y Autenticación

### Registro de Usuario:
1. El usuario completa el formulario de registro.
2. Se valida la información y se crea una nueva cuenta si es correcta.

### Inicio de Sesión:
1. El usuario ingresa sus credenciales en el formulario de inicio de sesión.
2. Se verifica la autenticidad de las credenciales.
3. Si son correctas y el usuario es administrador, se inicia sesión y se redirige a la interfaz de administrador.
4. Si son correctas y el usuario no es administrador, se inicia sesión y se redirige a la interfaz de usuario normal.
5. Si las credenciales son incorrectas, se muestra un mensaje de error.
